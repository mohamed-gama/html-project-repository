Applications of some programming languages

  • Applications of (C++) language

    1. Games: (C++) language overruns the complications, optimizes resource management and eases multiplayer with network.
    2. Web Browsers: The usage of (C++) is limited for scripting of websites and web applications. However, where speed and accuracy are wanted this language is still preferred.
    3. Advanced Computations and Graphics: (C++) language provides the means for building applications requiring real-time physical simulations, high performance image processing and mobile sensor applications.

  • Applications of (FORTRAN) language

    1. Number Crunching: (FORTRAN) is easy and efficient at processing mathematical equations because of the more natural way of expressing complicated mathematical functions and its quick execution time.
    2. Scientific, Mathematical, Statistical and Engineering Type Procedures: (FORTRAN) language is a good choice for these applications because of its rapid number crunching ability.

  • Applications of (SQl) language

    1. Data Integration Scripts: The major application of (SQL) is writing data integration scripts by the database developers.
    2. Retrieve Information: (SQL) language restores the subsets of information within a database for analytics applications and transaction processing. The most used elements of this language are select, insert, update, add, delete, create, truncate and alter.
