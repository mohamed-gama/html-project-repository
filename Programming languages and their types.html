Programming languages and their types

Some types of programming languages

Type About the type
Machine language This language consists of numeric codes for the processes that a specific computer can execute. The numeric codes are strings of binary digits which are converted from and to hexadecimal for human viewing and alteration.
Algorithmic languages These languages are designed to express mathematical and symbolic calculations. They can express algebraic operations in marking and give the usage of subprograms that package used operations for reuse.
Object-oriented languages These languages help to control in complication in large programs. Objects package data and the processes on them so that only the processes are publicly, accessible and the interior details of data structure are hidden.

Some programming languages

Language About the language
FORTRAN It is the first important algorithmic language. Its control structures included conditional (If) statements, recurrent loops (so called Do loops) and a (GoTo) statement that allowed nonsequential enforcement of program code.
SQL It is a language for assigning the organization of databases. Databases organized with (SQL) are called relational because this language provides an ability to inquiry a database for information that falls in a particular relation.
C++ This language has been one of the most important languages for education and industrial programming. Large parts of many operating systems like (Microsoft Corporation's windows 98) were written in this language.
