Advantages of some programming languages

  • (C++) language

    1. (C++) language presents the portability or platform independence which makes the user to run the same program on various operating systems easily.
    2. One of the best advantages of this language is the feature object-oriented programming which includes concepts such as classes, inheritance, polymorphism, data abstraction, and encapsulation which allow the code reusability.

  • (FORTRAN) language

    1. (FORTRAN) is legible and easy to understand.
    2. (FORTRAN) is like (MATLAB) but it has some mathematical processes.
    3. This language supports numerical analysis, scientific calculation, generic and structured programming.

  • (SQL) language

    1. (SQL) can retrieve large amount of data rapidly and efficiently. Processes such as Insertion, deletion and data manipulation are also done in almost no time.
    2. For data retrieval, big number of code lines is not wanted. All main keywords such as select, insert into and update are used and also the syntactical rules are not complicated in this language, which makes it the user-friendly language.
